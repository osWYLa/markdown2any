# 快速开始指南

## 5 分钟上手

### 1. 安装依赖

```bash
# 安装后端依赖
cd backend
npm install

# 安装前端依赖
cd ../frontend
npm install
```

### 2. 启动应用

**方式 A: 使用启动脚本 (推荐)**

```bash
./start.sh
```

**方式 B: 手动启动**

打开两个终端窗口:

终端 1 - 启动后端:
```bash
cd backend
npm start
```

终端 2 - 启动前端:
```bash
cd frontend
npm run dev
```

### 3. 访问应用

打开浏览器访问: http://localhost:5173

## 基本使用

### 第一次使用

1. 在左侧编辑器输入 Markdown 内容
2. 右侧会实时显示预览
3. 选择一个喜欢的主题 (简约明亮/深色优雅/温暖舒适)
4. 根据需要调整配置
5. 点击"生成图片"按钮
6. 等待 2-5 秒,图片生成完成
7. 点击"下载图片"保存到本地

## 调试技巧

### 后端服务检查

```bash
curl http://localhost:3001/api/health
# 应该返回: {"status":"ok"}
```

### 查看日志

后端日志会实时显示在终端中,包括:
- 请求信息
- 生成状态
- 错误信息

### 常见问题

**Q: 生成按钮点击后没反应?**
A: 检查浏览器控制台是否有错误,确保后端服务正在运行。

**Q: 生成的图片显示方块?**
A: 系统缺少中文字体,macOS 用户无需担心,Linux 用户需要安装中文字体。

**Q: 生成速度很慢?**
A: 首次生成可能较慢 (Puppeteer 启动),后续会快很多。
