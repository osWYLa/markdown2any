# 快速开始指南

## 5 分钟上手

### 1. 安装依赖

```bash
cd frontend
npm install
```

### 2. 启动应用

```bash
# 在根目录运行启动脚本
./start.sh
```

或者手动启动：

```bash
cd frontend
npm run dev
```

### 3. 访问应用

打开浏览器访问: http://localhost:5173

## 基本使用

### 第一次使用

1. 在左侧编辑器输入 Markdown 内容
2. 右侧会实时显示预览
3. 选择一个喜欢的主题 (简约明亮/深色优雅/温暖舒适)
4. 根据需要调整配置
5. 点击"生成图片"按钮
6. 等待 2-5 秒,图片生成完成
7. 点击"下载图片"保存到本地

### 示例内容

试试这个示例:

```markdown
# 今日分享

这是一个**精美的**文字图片生成器。

## 主要特点

- 支持多种主题
- 自定义样式
- 高清输出

*让你的文字更有影响力!*

[了解更多](https://example.com)
```

## 调试技巧

### 常见问题

**Q: 生成按钮点击后没反应?**
A: 检查浏览器控制台是否有错误信息。

**Q: 生成速度很慢?**
A: 纯前端生成通常在 1 秒内完成，如果内容特别复杂可能会稍慢。
